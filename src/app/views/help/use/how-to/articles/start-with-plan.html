我完全没有编程基础，如何选择课程？

课程的选择，首先需要确定自己所选的编程语言。

对于没有选定开发语言的学员，我们的建议是从 “Python” 语言开始，基于：

  1. Python 就业面相对较宽，在 “北上广深”，“成都”，“武汉”，“杭州”等有一定科技实力的地方，都有不错的岗位。
  2. 高级语言，比较易于 “入门”，而 Python 在高级语言中，又是属于比较容易 “入门” 的类型。
  3. 高级语言，在设计上有很强的 “通用性”，比如都包括了数据类型，数据结构，程序逻辑，开发框架等，学会一个，其它语言也较容易切入。

对于网站开发需求强烈的，也可以推荐 “PHP” 作为初学语言。如果还有不一样的需求，或疑问，请联系平台服务（邮箱，工单，QQ 群方式）。

选定编程语言后，对于没有编程经验的，或者只具有基本知识的，我们推荐相关编程语言的 “职业规划”，比如 Python 可以选择 “Python 软件工程师”，网页设计可以选择 “Web 前端工程师” 等，点击 这里 查看所有 “职业规划”。

对于 “职业规划” 还不满意，可以咨询平台服务，获取定制的私有 “职业规划”。
